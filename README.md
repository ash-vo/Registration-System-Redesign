# Academic System Interface Redesign

## Overview
This project focuses on redesigning key interfaces within a university system, including student registration, administrative tools, and academic advisor workflows. The goal was to improve usability, accessibility, and efficiency across multiple stakeholders.

## My Role
- Developed and prioritized user stories and backlog items based on stakeholder needs (students, administrators, academic advisors)  
- Managed tasks and tracked progress using Jira  
- Designed wireframes in Figma to visualize improved interfaces and workflows  
- Contributed to product decisions through agile iterations  

## Tools & Technologies
- Figma  
- Jira  
- Agile methodology  

## Process
- Analyzed stakeholder needs across different user groups  
- Identified usability challenges in existing systems  
- Created wireframes to improve navigation, accessibility, and workflow efficiency  
- Iterated on designs based on feedback  

## Key Improvements
- Simplified student course registration experience  
- Improved administrative task workflows and system efficiency  
- Enhanced advisor interfaces for better student tracking and decision-making  
- Increased accessibility and clarity across all user interfaces
